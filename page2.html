COIN COLLECTOR

P5.js Game

Coin Collector is a collection game that uses the P5 libraries. The game uses P5 to integrate gravity and velocity to maneuver the character. To control the character the player uses the W, A, and S keys to walk left, right, and jump. The goal of the game is to collect as many coins as you can in a certain amount of time. I enjoyed expanding upon the basic information that I've learned, to create a more complex game.
